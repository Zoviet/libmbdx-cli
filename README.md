# libmbdx-cli

Base and simple libmbdx cli frontent:

- Support put, del and get
- Commit changes
- Not support interactive mode
- Not support DUPKEYS

# Usage

```
mdbx [db_path...] [--help] [--del]  [--put] [--get] [--info] [--del]

```

Options:

--put key value 
--del key 
--get key 
--info  - database info
